- Python 3.8+
- Prefect
- requests

## Setup

1. Crea un entorno virtual (te recomiendo usar `conda`):
   ```bash
   conda create --name orch-env python=3.8
   ```
2. Activate the virtual environment:
   ```bash
   conda activate orch-env
   ```
3. Install requirements:
   ```bash
   pip install -r requirements.txt
   ```
